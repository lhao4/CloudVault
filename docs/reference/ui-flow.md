# CloudVault 云巢 — UI 流程文档

> **版本**：v2.0 | **状态**：已确认 | **平台**：Windows（Qt 6）

---

## 1. 设计规范

### 1.1 色彩系统

| Token | 色值 | 用途 |
|-------|------|------|
| `--accent` | `#3B82F6` | 主色：按钮、选中态、自己的消息气泡 |
| `--accent-hover` | `#2563EB` | 主色悬停 |
| `--accent-soft` | `#EFF6FF` | 主色浅底：选中列表项背景 |
| `--bg` | `#F4F6F8` | 应用背景 |
| `--surface` | `#FFFFFF` | 卡片/面板背景 |
| `--surface2` | `#F0F2F5` | 输入框、次级背景 |
| `--border` | `#E2E6EA` | 分割线、边框 |
| `--text` | `#111827` | 主要文字 |
| `--text2` | `#6B7280` | 次要文字 |
| `--text3` | `#9CA3AF` | 占位符、时间戳 |
| `--online` | `#22C55E` | 在线状态绿点 |
| `--danger` | `#EF4444` | 危险操作、错误状态 |

### 1.2 尺寸规范

| 组件 | 尺寸 |
|------|------|
| 登录窗口 | 400 × 自适应 |
| 主窗口 | 900 × 580（最小） |
| 侧边栏宽度 | 260px |
| 右侧详情面板宽度 | 220px |
| 圆角（大） | 12px |
| 圆角（小） | 8px |
| 输入框高度 | 44px |
| 图标按钮 | 30 × 30px |
| 头像（列表） | 40 × 40px |

### 1.3 字体

- 正文：`DM Sans`，回退 `PingFang SC`、`Microsoft YaHei`、`sans-serif`
- 等宽（日志）：`DM Mono`，回退 `monospace`

**交互说明**：

| 操作 | 触发条件 | 行为 |
|------|---------|------|
| 点击「登录」 | 字段非空 | 调用 AuthService::login()，按钮变灰禁用 |
| 点击「登录」 | 字段为空 | 输入框下方红色文字提示 |
| 登录成功 | — | 淡出关闭，打开主窗口 |
| 登录失败 | — | 输入框抖动动效 + 错误提示文字 |
| 点击「创建账号」 | 两次密码一致且满足长度 | 调用 AuthService::registerUser() |
| 注册成功 | — | 自动切回登录 Tab，提示"注册成功" |
| 服务器离线 | 启动时检测 | 绿点变红，文字变"连接失败" |

**状态机**：
```
空白 ──[输入]──► 可提交 ──[提交]──► 请求中（按钮禁用）
                                        ├──[成功]──► 跳转主界面
                                        └──[失败]──► 可提交 + 错误提示
```

## 6. 通用交互规范

### 6.1 加载状态

- 所有网络请求发出后，相关按钮立即禁用，文字变为「处理中…」
- 按钮禁用态：`opacity: 0.6`，`cursor: not-allowed`
- 文件上传/下载：底部显示进度条组件，含百分比 + 已传/总大小 + 取消按钮

### 6.2 错误提示

| 错误类型 | 提示方式 |
|---------|---------|
| 网络断开 | 主窗口顶部红色横幅："已断开连接，正在重连…"（3秒后自动重连） |
| 业务错误（用户不存在等） | 输入框下方红色小字提示，3秒后消失 |
| 输入格式错误 | 输入框边框变红 + 下方文字说明 |
| 操作成功 | 直接刷新列表，**不弹窗**（避免打断用户流） |
| 危险操作确认 | 使用系统弹窗确认（删除好友、退出群组等） |

### 6.3 通知规则

| 事件 | 通知方式 |
|------|---------|
| 收到新消息（当前会话） | 直接追加到消息区，自动滚底 |
| 收到新消息（其他会话） | 联系人列表角标 +1，系统托盘气泡 |
| 收到好友申请 | 立即弹窗确认（任意界面均触发） |
| 收到文件分享 | 立即弹窗确认 |
| 好友上/下线 | 静默更新列表在线状态，不打断用户 |

### 6.4 快捷键

| 快捷键 | 作用域 | 行为 |
|--------|--------|------|
| `Enter` | 登录界面 | 提交登录 |
| `Enter` | 聊天输入框 | 发送消息 |
| `Shift+Enter` | 聊天输入框 | 消息内换行 |
| `F5` | 全局 | 刷新当前面板 |
| `Delete` | 文件面板（已选中） | 删除选中项 |
| `F2` | 文件面板（已选中） | 重命名 |
| `Backspace` | 文件面板 | 返回上级目录 |
| `Ctrl+F` | 文件面板 | 聚焦搜索框 |
| `Ctrl+U` | 文件面板 | 上传文件 |
| `Escape` | 弹窗 | 关闭弹窗 |
