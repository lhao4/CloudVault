## 1. 设计基线

| Token | 色值 | 用途 |

## 3. 启动与导航流程

```text
应用启动
  ↓
App 创建 LoginWindow
  ↓
S-01 登录/注册页
  ↓ 登录成功
App 创建 MainWindow
  ↓
S-02 主窗口
  ├── 消息 Tab：SidebarPanel + ChatPanel + DetailPanel
  ├── 文件 Tab：SidebarPanel + FilePanel
  └── 我 Tab：ProfilePanel
```

登出流程：

```text
ProfilePanel 点击“退出登录”
  ↓
MainWindow 发出 logoutRequested
  ↓
App 销毁 MainWindow
  ↓
返回 LoginWindow
```

断线流程：

```text
TcpClient 断开
  ↓
App 通知 MainWindow 显示顶部红色横幅
  ↓
事件日志写入“连接已断开”
  ↓
若已加载服务器配置，则 3 秒后自动重连
```

## 4. S-01 登录 / 注册页

### 4.1 当前布局

- 顶部品牌头部
- 中间卡片容器
- `QTabWidget` 切换登录 / 注册
- 底部显示服务器连接状态

### 4.2 登录页字段

- 用户名
- 密码
- 登录按钮
- 服务器状态展示

当前已支持：

- 回车直接提交登录
- 连接成功、断开、错误状态实时刷新

### 4.3 注册页字段

- 用户名
- 昵称
- 密码
- 确认密码
- 创建账号按钮

当前限制：

- “昵称”字段仍保留在 UI 中
- 但注册请求当前只真正提交用户名和密码
- 昵称尚未进入协议、数据库和主界面资料体系

这意味着注册页的昵称目前是“界面采集但未持久化”的状态。

## 5. S-02 主窗口

### 5.1 整体结构

主窗口由四部分组成：

1. 顶部连接状态横幅
2. 中部 `QSplitter` 三栏内容区
3. 底部事件日志栏
4. 最底部导航栏

### 5.2 Tab 切换行为

#### 消息 Tab

- 显示左侧栏
- 显示聊天中栏
- 显示右侧详情栏

#### 文件 Tab

- 显示左侧栏
- 显示文件中栏
- 隐藏右侧详情栏

#### 我 Tab

- 隐藏左侧栏
- 显示个人页
- 隐藏右侧详情栏

## 6. S-03 左侧栏 `SidebarPanel`

### 6.1 当前用途

- 消息 Tab：联系人列表
- 文件 Tab：联系人列表仍保留，用于切换联系人文件空间

### 6.2 结构

- 顶部标题
- 搜索框
- 可选动作按钮
- 列表区

### 6.3 消息 Tab 状态

- 标题：`消息`
- 搜索占位：`搜索联系人…`
- 右上动作按钮可见，打开在线用户弹窗

### 6.4 文件 Tab 状态

- 标题：`文件`
- 搜索占位：`搜索联系人…`
- 右上动作按钮隐藏

### 6.5 列表项内容

每个联系人项当前展示：

- 头像首字母
- 用户名
- 最后一条消息预览
- 时间
- 未读数
- 在线状态

列表已按会话活跃度排序，并维护未读状态。

---

## 7. S-04 聊天面板 `ChatPanel`

### 7.1 当前结构

- 顶栏：头像、联系人名、在线状态、群组入口
- 中部：消息列表
- 底部：输入框、发送按钮

### 7.2 空状态

未选中联系人时，中栏显示空态提示，不显示伪造消息内容。

### 7.3 消息渲染

当前通过自定义绘制实现：

- 日期分割线
- 左右消息气泡
- 文件消息卡片样式

### 7.4 输入交互

- `Enter`：发送消息
- `Shift+Enter`：换行

### 7.5 群聊状态

当前点击群组入口会弹出 `GroupListDialog`。  
若选择群组，中栏切换到群聊占位页，但不会加载真实群消息。

换句话说：

- 群组入口已存在
- 群聊 UI 占位已存在
- 群聊业务尚未打通

## 8. S-05 文件面板 `FilePanel`

### 8.1 当前结构

- 顶部路径栏和工具按钮
- 搜索框
- 文件列表
- 传输进度行
- 底部操作栏

### 8.2 顶部按钮

当前已实现：

- 返回上级
- 上传
- 刷新
- 新建文件夹

### 8.3 文件操作

底部操作区当前已实现：

- 下载
- 分享
- 重命名
- 移动
- 删除

### 8.4 上传 / 下载行为

上传：

- 直接使用系统文件选择器 `getOpenFileNames`
- 支持多文件加入上传队列

下载：

- 直接使用系统保存对话框 `getSaveFileName`
- 选择单个文件的目标保存路径

### 8.5 搜索与快捷键

文件面板当前支持：

- 搜索框回车执行搜索
- `F5` 刷新
- `Delete` 删除
- `F2` 重命名
- `Backspace` 返回上级目录
- `Ctrl+F` 聚焦搜索框
- `Ctrl+U` 打开上传对话框

## 9. S-06 右侧详情面板 `DetailPanel`

当前用于展示：

- 联系人头像
- 联系人用户名
- 在线/离线状态
- 与该联系人的共享文件摘要

当消息 Tab 未选中联系人时，详情面板显示空态说明。

---

## 10. S-07 个人页 `ProfilePanel`

### 10.1 当前结构

- 中央卡片布局
- 头像首字母徽标
- 显示名
- `@username`
- 昵称输入框
- 个性签名输入框
- 保存 / 取消
- 退出登录

### 10.2 数据来源

- 资料草稿当前保存在本地 `QSettings`
- `保存`：写回本地草稿并刷新显示名
- `取消`：重新加载本地草稿

说明：

- 当前仍是本地资料页，不会同步到服务端
- 因此它更像“客户端个人偏好页”，不是跨设备统一资料系统

## 11. 弹窗流程

### 11.1 在线用户弹窗 `OnlineUserDialog`

入口：

- 消息 Tab 左侧栏顶部动作按钮

用途：

- 查看在线用户
- 选择用户并发起加好友

### 11.2 文件分享弹窗 `ShareFileDialog`

入口：

- 文件面板底部 `分享`

用途：

- 在好友列表中选择接收方
- 发送分享请求

### 11.3 群组列表弹窗 `GroupListDialog`

入口：

- 聊天面板顶栏群组按钮

现状：

- 已有独立弹窗和空态
- 支持选择群组名称并切到群聊占位页
- 没有真实群聊后端

## 12. 事件日志与网络反馈

### 12.1 顶部连接横幅

由 `MainWindow` 展示，`App` 驱动：

- 已断开连接，正在重连…
- 网络异常：xxx

### 12.2 底部事件日志栏

当前会记录：

- 登录成功
- 刷新好友
- 打开群组
- 上传开始 / 上传完成
- 下载完成
- 退出登录
- 重连与网络错误

事件日志支持展开 / 折叠。

## 13. 当前实现与旧设计稿差异

以下内容已经按现代码修正：

- `ProfileDialog` 改为 `ProfilePanel`
- 群聊从“完整界面”降级为“占位页 + 群组列表弹窗”

以下内容仍然不是完整实现：

- 注册昵称未真正入库
- 群聊业务未接通

## 14. 后续建议

如果继续沿当前 UI 结构推进，建议优先做：

1. 群聊消息链路和群组服务端
2. 注册昵称真正接入协议和资料页
3. 个人资料同步到服务端
4. 为客户端补自动化 UI 或集成测试
